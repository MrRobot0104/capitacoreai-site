14 Vulnerability Categories

Every scan covers the full attack surface — from hardcoded secrets to business logic flaws.

Secrets & Exposure

Hardcoded API keys, leaked .env files, service keys in client code.

Authentication

Missing auth on endpoints, weak JWT verification, brute-force exposure.

Authorization (IDOR)

User A accessing User B's data, missing ownership checks, broken RLS.

Injection Attacks

SQL injection, command injection, NoSQL injection, template injection.

Cross-Site Scripting

innerHTML with user input, unescaped AI output, DOM-based XSS vectors.

API Security

Missing rate limiting, wildcard CORS, SSRF, mass assignment.

Data Protection

PII in plaintext, missing encryption, insecure cookies and storage.

CSRF

Missing CSRF tokens, SameSite misconfig, cross-origin state changes.

File Upload & Handling

No type validation, path traversal in filenames, missing size limits.

Dependencies & Supply Chain

Known CVEs, outdated packages, missing lockfiles.

Infrastructure & Config

Debug mode in production, missing security headers, exposed admin routes.

Business Logic Flaws

Race conditions in payments, price manipulation, account enumeration.

Denial of Service

ReDoS patterns, unbound loops, missing pagination on list endpoints.

Logging & Monitoring

Missing audit trails, secrets in logs, no alerting for suspicious activity.

Critical High Medium Low
